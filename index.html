1. Vector Norms

A norm is a function $||\cdot||: \mathbb{R}^m \to \mathbb{R}$ that assigns a real-valued length to each vector that satisfy the following three conditions.
$$
\begin{aligned}
&(1)\, ||x|| \ge 0, \text{and } ||x|| = 0 \text{ only if } x = 0,\\
&(2)\, ||x + y|| \le ||x|| + ||y||,\\
&(3)\, ||\alpha x|| = |\alpha|\, ||x||.
\end{aligned}
$$
